# doroutine

[doroutine](https://doroutine.com) is a web application that let's you create routines and track the progress you make in the gym.

### Libraries used in the project include:

```
- typescript
- react
- redux
- redux-sagas
- react-router
- reselect
- styled-components
- jest
- puppeteer
```
